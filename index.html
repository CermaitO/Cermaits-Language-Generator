What is Kerbekulo?

Kerbekulo is a fictional language, made up purely for the purposes of this generator, which I made to practice JavaScript while indulging in my love of languages. Kerbekulo has a fixed set of affixes and a fixed morphophonology, however I have not created any words because that would take the fun away from this generator! Here, you can create the words yourself and see how they inflect and compound together. First, enter the words and then read how the fit into the language!
